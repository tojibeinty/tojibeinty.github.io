# 🧬 Bio CV - موقعك البيولوجي الشخصي

## الملفات

| الملف | الوظيفة |
|-------|---------|
| `index.html` | الموقع الرئيسي |
| `admin.html` | لوحة الإدارة |
| `bot.js` | بوت تلغرام |

---

## 🚀 تشغيل الموقع

### محلياً:
فقط افتح `index.html` في المتصفح.

### على استضافة:
ارفع الملفات على أي استضافة (Netlify مجاناً).

---

## ⚙️ إعداد لوحة الإدارة

1. افتح `admin.html`
2. من "الملف الشخصي" غيّر اسمك وبياناتك
3. من "الحالات" أضف حالاتك
4. من "الفيديوهات" أضف فيديوهاتك

---

## 🤖 إعداد بوت تلغرام

### الخطوات:
1. اذهب لـ [@BotFather](https://t.me/BotFather) في تلغرام
2. أرسل `/newbot` واختر اسماً للبوت
3. انسخ **التوكن**
4. اذهب لـ [@userinfobot](https://t.me/userinfobot) لمعرفة **Chat ID** الخاص بك
5. افتح `bot.js` وضع:
   - `TOKEN = 'التوكن_هنا'`
   - `ADMIN_ID = رقمك_هنا`

### تشغيل البوت:
```bash
npm install node-telegram-bot-api
node bot.js
```

### أوامر البوت:
| الأمر | الوظيفة |
|-------|---------|
| `/addcase` | إضافة حالة (مع صورة) |
| `/addvideo` | إضافة فيديو |
| `/listcases` | عرض الحالات |
| `/listvideos` | عرض الفيديوهات |
| `/deletecase 1` | حذف الحالة رقم 1 |
| `/deletevideo 1` | حذف الفيديو رقم 1 |

---

- البيانات تُخزن في `localStorage` للمتصفح (للبوت: `data.json`)
